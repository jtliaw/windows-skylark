## 🚀 核心特性

### 🎯 智能翻译系统
- **⚡ 多引擎融合** - Google/DeepL/百度/微软 (在线)
- **🧠 路径优化算法** - 优先使用直接翻译，提升速度和准确率

### 🖱️ 极简操作体验
- **框选翻译** - 左键拖拽框选，右键双击翻译
- **一键隐藏** - 左键单击显示/隐藏翻译窗口
- **滚轮浏览** - 鼠标滚轮查看完整翻译内容

### 🔧 强大的引擎支持

Skylark 在线翻译服务，适用于更高精度或更多语言方向的翻译需求：
| 平台           | 是否需要 API 密钥     | 注册/获取地址                                                                           |
| ------------ | --------------- | --------------------------------------------------------------------------------- |
| 🌐 Google 翻译 | ❌ 无需密钥          | 内置公共接口，无需注册                                                                       |
| 🌐 DeepL 翻译  | ✅ 需要 API 密钥     | [DeepL Pro 注册](https://www.deepl.com/pro-api)                                     |
| 🌐 百度翻译      | ✅ 需要 APP ID 与密钥 | [百度翻译开放平台](https://fanyi-api.baidu.com/)                                          |
| 🌐 微软翻译   | ✅ 需要 API 密钥      | [注册获取密钥（Azure Portal）](https://azure.microsoft.com/products/cognitive-services/translator/) |

## ⚡ 快速开始

### 📦 无需安装
下载后解压直接双击SkylarkTranslator.exe即可运行

### 🎯 首次配置
1. **启动软件** - 点击SkylarkTranslator.exe图标
2. **安装语言包** - 进入【语言包管理】，选择需要的翻译的语言
3. **开始使用** - 框选屏幕区域，右键双击翻译，左键则是隐藏/显示翻译框！
4. **谷歌api** - 谷歌api能够自动帮你选出最快翻译的站点！

### 📦 删除软件
要删除软件可以直接删除整个文件夹即可，唯独Tesseract OCR的语言包是下载到用户的文件夹内，需要使用软件内的删除Tesseract OCR语言包的功能删除！
